Por que foi criado?

O JavaScript, a princípio, foi criado com o objetivo principal de validar formulários HTML. Na verdade, o JavaScript foi concebido para rodar no servidor. No entanto, com o passar do tempo isso foi modificado e, hoje, o JavaScript é uma linguagem Client-side.

Quando foi criado o javascript?

Origem e História. Concebido como uma linguagem do lado do servidor por Brendan Eich (então empregado pela Netscape Corporation), o JavaScript logo chegou ao Netscape Navigator 2.0 em setembro de 1995.

Curiosidade

Ele habilita páginas da web interativas e, portanto, é uma parte essencial dos aplicativos da web. A maioria dos sites o usa e todos os principais navegadores da web têm um mecanismo JavaScript dedicado para executá-lo. JavaScript é de encadeamento único.

O que é JavasScript?

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Características do JavaScript

Javascript é uma linguagem baseada em objetos. O projeto de programas orientados a objetos tenta tratar um programa como um conjunto de partes individuais (objetos) que executam tarefas distintas, e não como uma seqüência de comandos que executam uma tarefa específica. Objetos similares são agrupados em classes.

Importância do JavaScript

O JavaScript permite inserir vários efeitos, fazendo com que o site fique mais dinâmico, principalmente, porque permite ao programador da web melhorar a página do site. A linguagem de programação JacaScript nos permite executar instruções como resposta às ações do usuário.
